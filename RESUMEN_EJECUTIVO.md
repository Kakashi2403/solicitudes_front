## 🚀 TU API ESTÁ INTEGRADA

**Endpoint:** `https://localhost:7298/api/Token`
**Método:** `POST`
**Respuesta:** `{ body: { accessToken }, errors, warnings }`

**Automaticamente:**
- ✅ Se decodifica el JWT
- ✅ Se extrae: id, email, fullName
- ✅ Se guarda en localStorage
- ✅ Se valida en el backend
