```
Usuario Ingresa Credenciales
         ↓ (FormComponent)
   AuthService.login()
         ↓ (LoginUseCase)
   HttpAuthAdapter.login()
         ↓ (HTTP POST)
API: https://localhost:7298/api/Token
         ↓ (Response JSON)
   LoginMapper.decodeJwt()
         ↓ (Extrae datos del JWT)
   LogicStorage.saveUser() + saveToken()
         ↓ (localStorage)
   LoginPageComponent muestra UserInfoComponent
         ↓ (UI Update)
```

## ✨ CARACTERÍSTICAS DESTACADAS

✅ **Decodificación automática de JWT**
   - El token se decodifica sin validación (válida en backend)
   - Extrae automáticamente: id, email, fullName
   - Calcula expiración dinámicamente

✅ **Manejo de errores robusto**
   - CORS errors → mensaje amigable
   - SSL errors → solución incluida
   - Credenciales inválidas → feedback claro
   - Codificación de JWT inválida → manejo graceful

✅ **Persistencia inteligente**
   - Usuario se guarda en localStorage
